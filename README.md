# CodeAlpha_Credit_Scoring
Credit Scoring Machine Learning project developed using Python and Logistic Regression. Predicts loan approval based on financial data. Created for CodeAlpha ML Internship.
# CodeAlpha Credit Scoring Model

## 📌 Project Overview

This project predicts whether a person's loan will be approved or rejected using Machine Learning.

The model is trained using financial details such as age, income, loan amount, and credit score.

This project is developed as part of the CodeAlpha Machine Learning Internship.
